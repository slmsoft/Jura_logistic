# Dashboard Management Tracking

Vue.js приложение для отслеживания доставок, реализованное по дизайну из Figma.

## Установка

```bash
npm install
```

## Запуск проекта

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка для продакшена

```bash
npm run build
```

## Структура проекта

```
src/
├── components/
│   ├── Sidebar.vue      # Боковая панель навигации
│   ├── DeliveryCard.vue # Карточка доставки
│   └── Map.vue          # Компонент карты
├── App.vue              # Главный компонент
├── main.js              # Точка входа
└── style.css            # Глобальные стили
```

## Особенности

- Точная реализация дизайна из Figma
- Адаптивная верстка
- Компонентная архитектура Vue 3
- Современный UI с анимациями
